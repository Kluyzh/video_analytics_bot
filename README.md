# Telegram Bot for Video Analytics

Бот для аналитики видео, преобразующий запросы на естественном языке в SQL-запросы.

## Архитектура

1. **Telegram Bot** (aiogram 3.x) - принимает сообщения пользователей
2. **LLM Handler** (LLM API) - преобразует текст в SQL
3. **PostgreSQL** - хранит данные о видео и снапшотах
4. **Asyncpg** - асинхронное взаимодействие с БД

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте файл `.env` на основе `.env.example`
3. Запустите:
```
docker-compose up --build
```
